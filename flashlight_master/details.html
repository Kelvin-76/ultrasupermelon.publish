Flashlight Master is one of the best flashlight applications on the market
*No pop-up ads
*Contains every flashlight function you want in one package
*Free of charge

=====================

Features
-Simple ON/OFF light
-Light sensor toggled flashlight - turn on when environment is dark
-Strobe light (frequency configurable)
-SOS (frequency configurable)
-Bright Screen light
-Editable Morse Code signal
